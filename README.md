# Diabetes Prediction AI

## Overview

Diabetes Prediction AI is a Machine Learning web application that predicts whether a patient is at a high or low risk of diabetes based on medical parameters.

The application is built using Python, Scikit-learn, Flask, HTML, and CSS.

---

## Features

- Predicts diabetes risk using a trained Machine Learning model
- User-friendly web interface built with Flask
- Displays prediction confidence
- Responsive and clean UI
- Educational disclaimer included

---

## Installation

Clone the repository

```bash
git clone https://github.com/harinisharma-t/DiabetesPredictionAI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app/app.py
```

Open

```
http://127.0.0.1:5000
```
